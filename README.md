# Toy Calculator 🍒

A simple calculator built with HTML, CSS, and JavaScript.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Decimal numbers
- Clear button
- Cute pastel UI

## How to Use

1. Enter a number.
2. Choose an operator (+, -, *, /).
3. Enter another number.
4. Press = to see the result (or continue calculation).
5. Press C to clear the calculator.
